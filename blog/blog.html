1.Discuss the scope of var, let, and const?

Ans:Var: No Block Scope: Variables declared with var are not block-scoped, meaning they can "leak" out of if statements or loops. let: Block Scope: Variables declared with let are block-scoped. They are only visible within the block in which they are declared, such as if statements or loops. const: Cannot Be Re-assigned: Variables declared with const cannot be reassigned.

2.Tell us the use cases of null and undefined?

Ans: Null : in JavaScript means an empty value and is also a primitive type in JavaScript. The variable which has been assigned as null contains no value. Undefined : on the other hand, means the variable has been declared, but its value has not been assigned.

3.What do you mean by REST API?

Ans: RESTful APIs are commonly used in web development and are known for their simplicity, scalability, and ease of use. They make use of standard HTTP protocols, which are widely supported by a variety of programming languages and platforms. This makes REST a popular choice for building APIs that can be accessed from a wide range of clients, including web browsers, mobile apps, and other applications.
